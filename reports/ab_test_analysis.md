# A/B Test Analysis Report

## Executive Summary

Local-Helix 추천 시스템의 A/B 테스트 시뮬레이션 결과를 분석한 보고서입니다.

**테스트 기간**: 2026-01-14  
**총 참여 유저**: 1,000명  
**테스트 방법**: 50:50 무작위 그룹 할당

---

## Test Groups

### Group A (Control) - 511명
- **전략**: 인기 상품 Top 5 추천 + 랜덤 발송 시간 (9-21시)
- **CTR**: 78.08%
- **평균 만족도**: 3.53/5
- **평균 구매 수**: 1.56개

### Group B (Test) - 489명
- **전략**: ML 모델 추천 + 최적 발송 시간
- **CTR**: 74.03%
- **평균 만족도**: 3.63/5
- **평균 구매 수**: 1.47개

---

## Key Findings

### 1. Click-Through Rate (CTR)
- **Group A**: 78.08%
- **Group B**: 74.03%
- **차이**: -4.05% (Group B가 낮음)
- **CTR Lift**: -5.19%

**통계적 유의성**: 
- Chi-square test: χ² = 2.0401, p = 0.1532
- **결론**: 통계적으로 유의하지 않음 (p ≥ 0.05)

### 2. User Satisfaction
- **Group A**: 3.53/5
- **Group B**: 3.63/5
- **차이**: +0.10 (Group B가 높음)

**통계적 유의성**:
- T-test: t = -1.5089, p = 0.1316
- **결론**: 통계적으로 유의하지 않음 (p ≥ 0.05)

### 3. Purchase Count
- **Group A**: 1.56개
- **Group B**: 1.47개
- **차이**: -0.09개 (Group B가 낮음)

**통계적 유의성**:
- T-test: t = 1.2683, p = 0.2050
- **결론**: 통계적으로 유의하지 않음 (p ≥ 0.05)

---

## Interpretation

### 주요 인사이트

1. **CTR 역전 현상**
   - ML 모델 추천(Group B)이 인기 상품 추천(Group A)보다 CTR이 낮게 나타남
   - 예상과 다른 결과이며, 다음 원인들을 고려해야 함:
     - 인기 상품의 강력한 대중성
     - ML 모델의 개인화가 아직 충분하지 않을 가능성
     - 시뮬레이션 페르소나의 특성

2. **만족도는 향상**
   - Group B의 만족도가 약간 높음 (3.63 vs 3.53)
   - 개인화된 추천이 사용자 경험에는 긍정적 영향
   - 통계적으로 유의하지는 않지만 긍정적 신호

3. **통계적 유의성 부족**
   - 모든 지표에서 p-value > 0.05
   - 샘플 크기(1,000명)가 충분하지 않거나
   - 실제로 두 그룹 간 차이가 크지 않을 가능성

---

## Recommendations

### 단기 개선 방안

1. **샘플 크기 증가**
   - 현재 1,000명 → 5,000명 이상으로 확대
   - 통계적 검정력(power) 향상

2. **ML 모델 개선**
   - Feature Engineering 재검토
   - 하이퍼파라미터 튜닝
   - 더 많은 학습 데이터 활용

3. **하이브리드 전략 고려**
   - 인기 상품 + ML 추천 혼합
   - 예: Top 2 인기 상품 + Top 3 개인화 추천

### 장기 개선 방안

1. **실제 유저 데이터 수집**
   - 시뮬레이션이 아닌 실제 A/B 테스트 진행
   - 실제 사용자 행동 패턴 분석

2. **다양한 세그먼트 분석**
   - 나이대별, 예산별 세분화 분석
   - 각 세그먼트에 최적화된 전략 수립

3. **추가 지표 수집**
   - 전환율(Conversion Rate)
   - 평균 주문 금액(AOV)
   - 재방문율(Retention)

## Conclusion

현재 A/B 테스트 결과, ML 기반 추천 시스템이 인기 상품 추천 대비 명확한 우위를 보이지 않았습니다.
